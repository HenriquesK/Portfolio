PPE 1 : Réservation d'un séjour

Durant ce projet, nous avons travaillé avec le framework Code Igniter qui utilise une interface MVC (Modèle, Vue, Controller). Ma mission étais à partir d'un formulaire fournis, récupérer les données saisis puis les rentrer dans la base de donnée. Tous ça suivant l'interface du framework. De plus j'ai aidé à l'éllaboration de la base de donnée grâce à un diagramme de classe puis grâce à des requêtes SQL.
